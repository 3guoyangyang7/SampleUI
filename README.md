# 中科世为 FlyThings 控件集演示程序
## FlyThings 简介
FlyThings是中科世为针对物联网推出的一套带完整GUI，可以快速完成项目开发的一套小型操作系统。
系统特性：
1. 基于Linux深度裁剪优化
2. 自主的Framework部件
3. 具备界面堆栈管理
4. 具备网络API
5. 外部通讯串口API
6. 专用的GUI开发工具和源码开发工具
7. 支持自定义控件
